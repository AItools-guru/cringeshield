🚀 How to Use CringeShield (Step-by-Step)
1
Input Your Draft or Select a Template
Paste your raw thoughts or rough bullet points into the Post Editor. Or click any Quick Start Template chip (Meeting Sync, Tech Spec, Job Rejection, AI Launch) to test sample drafts.
2
Use Quick Utility Tools & Hook Vault
Click ⚡ Auto-Space for mobile line breaks, 🔗 Clean Lists to strip list hyphens, ❌ De-Buzz to clean AI jargon locally, or click 💡 Hooks to open the Viral Hook & CTA Vault.
3
Select Copywriting Formula & Tone Persona
Choose a copywriting framework (Hook-Value-CTA, PAS, AIDA, Contrarian Take) and persona tone (Pragmatic Builder, Storyteller, Data Critic, Thought Leader) to guide the AI refactoring engine.
4
Click "Shield Post" & Inspect Output Tabs
Click Shield Post to generate clean copy. Switch between right-pane tabs:
  • Analyzer: Dynamic Cringe Gauge (0-100%), Buzzword radar, and A11y health index.
  • Preview: Test pixel-perfect mobile post rendering and interactive Like counters.
  • Before & After: Compare raw draft vs shielded output with color-coded diff highlights.
  • Carousel PDF: Preview 4:5 ratio slide cards across 5 visual themes and export multi-page PDFs!
5
Manage Workspace Drafts
Click 📁 Drafts in the top navigation bar to auto-save, load, delete, or export your workspace post drafts to JSON files.
⚙️ Settings & Security Architecture
🔑 Gemini API Key & Simulator Mode
Click API Config in the header to enter your Google Gemini API Key (gemini-2.5-flash). If no key is configured, CringeShield automatically operates in zero-configuration Simulator Mode, applying offline formatting rules without needing API access.
👤 Preview Profile Customization
Configure your preview author name, headline, avatar image URL, and toggle the Verified Creator Badge inside the Config drawer to see your post under your personal branding.
🔒 Privacy & Security Architecture
CringeShield is 100% Client-Side. Your API key and drafts are stored strictly in your browser's private localStorage. No draft text or API keys are ever transmitted to any tracking database or middleman server.
